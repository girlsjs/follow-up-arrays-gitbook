## Funkcje wewnątrz obiektów
Funkcje wewnątrz obiektów nazywamy **metodami**. Tworzy się je tak:
```javascript
let myObject = {
  sayHi: function () {
    console.log('Hi!')
  }
}
```

A wywołuje tak:

```javascript
myObject.sayHi();
// Wypisze: Hi!
```
Ale o funkcjach innym razem :) Na następnym follow-upie zajmiemy się tematyką funkcji. Dziś wracamy do obiektów.
